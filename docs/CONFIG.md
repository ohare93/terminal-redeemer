# Configuration

Configuration schema will be implemented incrementally as part of V1.

Initial planned keys:

- `stateDir`
- `profile`
- `capture.enabled`
- `capture.interval`
- `capture.snapshotEvery`
- `retention.days`
- `processMetadata.whitelist`
- `processMetadata.whitelistExtra`
- `restore.appAllowlist`
- `restore.terminal.command`

Canonical behavior is defined in `DESIGN.md`.
